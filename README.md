Test from client-server app with aiohttp.

This project use:
    Python 3.10.5

1.- Create env:
    py -m venv env

2.- Activate env (Windows) :
    env\Scripts\activate

3.- Install libraries:
    pip install -r requirements.txt
    
4.- Run server:
    py server.py

5.- In other terminal with the same env activated, run client:
    py client.py
